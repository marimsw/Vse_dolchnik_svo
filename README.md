Анализировать полученный файл с репозитория Smerti_SVO_Stop_Fraza_JSONL_domen
Быстрый скрипт для извлечения записей с группой "Должник СВО" из JSONL файла и сохранения в Excel.
Скрипт выполняет одну простую задачу:
+ Читает большой JSONL файл (JSON Lines format) построчно
+ Фильтрует записи, у которых поле группа равно "Должник СВО"
+ Сохраняет отфильтрованные данные в Excel файл (.xlsx)

Как работает
+ Запрос пути к JSONL файлу от пользователя
+ Построчное чтение (не загружает весь файл в память целиком)
+ Парсинг каждой строки как отдельного JSON объекта
+ Проверка условия: data.get('группа') == "Должник СВО"
+ Сбор подходящих записей в список
+ Создание Pandas DataFrame и сохранение в Excel

Особенности:
+ Прогресс-индикатор - показывает количество обработанных строк и найденных записей
+ Подсчет времени выполнения
+ Обработка ошибок - пропускает битые JSON строки
+ Поддержка прерывания (Ctrl+C)

Производительность
+ Обрабатывает миллионы строк за короткое время
